# 참고할 만한 기능:

### Web Crawling:  연합뉴스 뉴스 코너를 크롤링해 캐러셀 생성

[연합뉴스 뉴스 코너](https://www.yna.co.kr/news)

### State Management(Context) :  컨텍스트를 사용한 상태관리, 구글 로그인 지원

### React Router : 라우터를 사용한 페이지 간 이동

### DB 연동 : MariaDB 사용, 게시판 및 댓글과 대댓글 기능 구현
